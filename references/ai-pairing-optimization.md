# AI + Human Collaboration Workflow
## How to use Antigravity (or any Agent) to build Awwwards-level sites

Building a SOTY-level site with an AI agent requires a specific "Director" mindset rather than a "Coder" mindset.

---

## 1. THE "DIRECTOR" PROTOCOL

Don't ask the AI to "Build a cool site". Use the **Archetype System**.

### Bad Prompt: 
"Make a dark portfolio with animations."

### Professional Prompt (using this skill):
"Trigger awwwards-website. Use Archetype 03 (Experimental Studio). Palette: Deep Dark. Signature Moment: A clip-path image reveal on scroll. Target Output: Next.js + GSAP."

---

## 2. ITERATIVE REFINEMENT
Creative development is 10% structural and 90% polish.

1. **Phase A (Structural):** Let the AI build the layout and core GSAP timelines.
2. **Phase B (Motion Review):** Inspect the durations. Use the **"Slower is More Premium"** rule. Ask: "Double the duration of all entrance animations and use power4.out".
3. **Phase C (The Secret Layer):** Ask for specific "Signature Techniques" (Phase 5 of the skill) one by one.

---

## 3. DEBUGGING VISUALS
AI is great at code, but YOU have the eyes.

- **Check for "Mechanical Feel":** If animations look robotic, ask for "Randomized staggers" or "Elastic easing".
- **Check for "Visual Density":** If the site feels "busy", use the **Negative Space Protocol** (Phase 2).
- **Check for "Gaps":** "Build the about page to match the homepage's design system exactly."

---

## 4. CODE MAINTENANCE
Awwwards sites often have "Spaghetti Motion" (hundreds of GSAP lines). 

- Ask the AI to: "Componentize the animations into reusable hooks (e.g., `useReveal`)".
- Use **CSS Variables** for everything that might change (colors, timings, paddings).

---

## 5. THE AI DESIGN PARTNER (generate_image)
Don't wait for a designer. Use the agent to create assets:

- "Generate a minimalist architectural photograph in high-contrast B&W for the hero background."
- "Generate a series of 4 abstract glass-bubble icons for the services section."

---

## 6. FINAL POLISH CHECKLIST (Ask the Agent)
"Perform a Phase 8 Verification Audit on the current build."
- Accessibility?
- 60 FPS performance?
- Mobile typography scale?
- OpenGraph tags?
